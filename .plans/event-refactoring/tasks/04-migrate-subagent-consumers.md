# T-004: Migrate `instance.ts` and `controller.ts`

**Phase**: subagents  
**Dependencies**: T-003  
**Requirements**: FR-006, NFR-005

## Objective

Migrate all `.emit()` calls in `packages/subagents/src/instance.ts` and `packages/subagents/src/controller.ts` to use the new `.publish(creator, payload)` API.

## Changes in `instance.ts`

Every `this.eventBus.emit({ type: ..., ... })` becomes a structured publish:

Current:
```typescript
this.eventBus.emit({
  type: "task.start",
  instanceId: this.instanceId,
  taskId: this.taskId,
  definitionName: this.definitionName,
  description: this.taskDescription,
  timestamp: now,
});
```

New:
```typescript
this.eventBus.publish(SubAgentEvents.taskStart, {
  instanceId: this.instanceId,
  taskId: this.taskId,
  definitionName: this.definitionName,
  description: this.taskDescription,
});
```

Note: `timestamp` is auto-generated by `publish()`, no need to pass it explicitly.

## Changes in `controller.ts`

Current:
```typescript
this.eventBus.on("*", options.onEvent);
this.eventBus.on("lifecycle.change", options.onLifecycleChange);
this.eventBus.on("task.end", async (event) => { ... });
```

New:
```typescript
const unsub = this.eventBus.subscribe(options.onEvent);           // wildcard
const unsub2 = this.eventBus.subscribe(
  SubAgentEvents.lifecycleChange, 
  options.onLifecycleChange
);
this.eventBus.subscribe(SubAgentEvents.taskEnd, async (event) => { ... });
```

And for inline emit:
```typescript
this.eventBus.emit(event);  →  this.eventBus.publish(SubAgentEvents.taskEnd, event.result);
```
